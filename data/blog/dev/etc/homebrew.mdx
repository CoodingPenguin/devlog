## 설치

### xcode command line tools 설치

```bash
xcode-select --install
```

### Homebrew 설치

내가 사용하는 MacBook의 칩이 어떤 칩인지에 따라 설치 방법이 다르다.

#### Intel 칩인 경우

```bash
# Intel 칩인 경우
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

#### Apple Silicon 칩인 경우

```bash
# arm64 터미널인 경우
arch -arm64 /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

# x86_64 터미널인 경우
arch -x86_64 /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

# homebrew 환경 설정
cat <<'EOF' >> ~/.zprofile
CPU=$(uname -m)
if [[ "$CPU" == "arm64" ]]; then
  eval "$(/opt/homebrew/bin/brew shellenv)"
else
  export PATH=/opt/homebrew/bin:$PATH
  eval "$(/usr/local/bin/brew shellenv)"
fi

alias ibrew="arch -x86_64 /usr/local/bin/brew"
alias abrew="arch -arm64 /opt/homebrew/bin/brew"

EOF
```

## 명령어

### brew를 최신 버전으로 업데이트

```bash
brew update
# Updated 3 taps (homebrew/cask-fonts, homebrew/core and homebrew/cask).
# ==> New Formulae
# actions-batch                kubeshark                    pkl
# ...
```

### 패키지 업그레이드

`brew update`가 선행된다.

```bash
# 전체 패키지 업그레이드
brew upgrade
# ==> Upgrading 97 outdated packages:
# cjson 1.7.16 -> 1.7.17
# coreutils 9.1 -> 9.4
# gmp 6.2.1_1 -> 6.3.0
# ...

# 인자로 넘긴 패키지만 업그레이드
brew upgrade <패키지명1> <패키지명2> <패키지명3>
```

### 업그레이드가 필요한 패키지 확인

```bash
brew outdated
# aom (3.7.0) < 3.8.1
# autoconf (2.71) < 2.72
# bat (0.22.1) < 0.24.0
# ...
```

### 패키지 검색

```bash
brew search <패키지명>
# ==> Formulae
# app-engine-python            python-kiwisolver            python-tabulate
# ...
```

### 패키지 설치

```bash
brew install <패키지명>[@버전]
# ==> Downloading https://ghcr.io/v2/homebrew/core/python/3.12/manifests/3.12.2_1
# ...
```

### 설치된 패키지 확인

```bash
brew [list | ls]
# ==> Formulae
# aom		gnupg		libtermkey	msgpack		rav1e
# ...

# 더 간단히
brew leaves
# automake
# bat
# cmake
# ...
```

### 패키지 정보 확인

```bash
brew info <패키지명>
# ==> bat: stable 0.24.0 (bottled), HEAD
# Clone of cat(1) with syntax highlighting and Git integration
# https://github.com/sharkdp/bat
```

### 패키지 삭제

```bash
brew uninstall | remove | rm <패키지명>
# Uninstalling /usr/local/Cellar/python@3.7/3.7.16... (4,847 files, 76.7MB)
```

### 최신 버전 외 버전의 패키지 삭제

```bash
brew cleanup <패키지명>
```

### 패키지 종속성 확인

```bash
brew deps --tree --installed <패키지명>
# bat
# ├── libgit2
# │   ├── libssh2
# │   │   └── openssl@3
# │   │       └── ca-certificates
# │   └── openssl@3
# │       └── ca-certificates
# └── oniguruma
```
