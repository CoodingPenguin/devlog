## RNN 속 수학

시퀀스 데이터가 뭘까? 소리, 문자열, 주가 등의 데이터
시퀀스 데이터는 i.i.d.를 잘 위배하기 때문에 순서를 바꾸거나 과거 정보에 손실이 발생하면 데이터의 확률분포도 바뀐다
-> 순서를 마음대로 바꾸면 X
-> 앞뒤 맥락 없이 미래 예측 X
그만큼 순서가 중요하다.중요하다

시퀀스 데이터는 어떻게 다루나 -> 조건부 확률!
베이즈 법칙을 이용해 예측한다.

하지만 그렇게 하면 값이 엄청나게 커진다.
항상 모든 과거의 정보만 보는 것은 아니다.
시퀀스 데이터를 다루기 위해선 길이가 가변적인 데이터를 다룰 수 있는 모델이 필요하다.
고정된 길이 즉, 일부 과거 정보만 보는 경우 AR(\gamma) 자기회귀모델이라고 부른다.
이 때, 바로이전 정보를 제외한 나머지 과거를 잠재변수 H_t로 나타낼 수 있으며 과거의 정보를 인코딩한 결과이다.

이 H_t를 신경망을 통ㅇ해 구한다. -> 잘 보면 RNN, LSTM에서 본게 보인다.

RNN 이해
mlp와 유사
이전 순서의 잠재변수와 현재의 입력을 이용하여 모델링
잠재변수와 연결그래프에 따라 순차적으로 계싼 (역전파) = BPTT

BPTT
미분의 곱으로 이루어진 항
여기서 보면 수식은 어려워..
여기서 그 부분이 길어지면 이 항은 불안정해져? -> 다 곱이어서 미분값이 엄청 커지거나 작아져
-> 기울기를 끊는 것이 필요하다 trucated BPTT (lstm, gru)

### RNN (Recurrent Neural Network)

단점: 과거 정보를 취합해서 미래에서 고려해야 하는데 fixed rule 때문에 너무 오래된 과거는 미래에 살아남기 힘듦 memento
왜? 활성화 함수가 sigmoid tanh라고 하면 계속 압축이 될 텐데 계속 반복되면서 가장 처음의 정보는 점차 사라짐
relu라면? relu는 양수면 기존의 값을 가져가기 때문에 계속 곱해지면서 값이 커져 exploding gradient 문제가 발생함
