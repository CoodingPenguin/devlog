기계학습에서 사용되는 손실함수(loss function)의 작동 원리는 데이터 공간을 통계적으로 해석해서 유도한다.

회귀분석의 손실함수인 L2-Norm은 예측 오차의 분산을 가장 최소화하는 방향으로 학습한다. 저번 식 가져와서 생각해보자.
왜 예측 오차의 분산일까?

확률분포
데이터 공간을 xxy즉, 확률변수 x와 y로 이루어진 데이터 공간이며, D는 데이터 공간에서 데이터를 추출하는 분포를 말한다. 즉, 확률분포

데이터 = 확률변수로 (x, y) ~ D로 표기
확률변수 : 표본 공간 S에서 실수 R로의 함수 X:S -> R을 확률 변수라고 한다. 즉, 어떤 경우를 그에 대응하는 실수로 만드는 함수이ㅏㄷ.
D는 데이터 분포 예를 들면 정규분포, 포아송 분포 이런 것

확률변수는 확률분포에 따라 이산형과 연속형 확률변수로 구분된다.

- 이산형 확률변수 예: 베르누이 확률변수, 이항확률변수, 푸아송 확률변수
- 연속형 확률변수 예: 균일 확률변수, 정규 확률변수, 지수확률변수

이산형확률변수는 확률변수가 가질 수 있는 모든 경우의 수를 고려하여 확률을 더해서 모델링을 한다.

pmf
cmf
기댓값, 분산

연속형확률변수는 데이터 공간에 정의된 확률변수의 밀도 위에서의 적분을 통해 모델링한다. 밀도 = 누적확률분포(cdf)의 변화율을 모델링한 것이며 절대로 확률로 해석하면 안된다.

pdf
cdf

결합분포는 확률분포 D를 모델링한다?
P(x, y)는 x와 y의 결합되 ㄴ확률분포이다.
P(X)는 입력 X에 대한 주변확률분포로 Y에 대한 정보는 없다.
X 따로, Y따로 해서 결합시켜야
결합분포에서 주변확률분포 유도가 가능하다 (이건 확통 강의노트 참고 다시 공부하자)

조건부확률 p(y|x)는 입력변수 x에 대해 정답이 y일 확률을 의미한다.
연속확률분포에서는 밀도로 해석

로지스티 회귀 -> 데이터에서 추출된 패턴을 기반으로 확률을 해석
분류 문제 -> 특징패턴고 ㅏ가중치 행렬을 통해 조건부 확률 계싼

회귀 문제 -> 조건부 기대값을 추청 거기서 조건부 확률이 사용되는데 이거는 엠제이님 이슈 참고
