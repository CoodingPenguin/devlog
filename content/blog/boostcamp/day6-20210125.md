## 벡터

벡터가 무엇인가?

1. **숫자**를 원소로 갖는 배열을 말한다.
2. 공간에서 한 점을 나타낸다. (기하학적)
3. 벡터는 원점으로부터 상대적 위치를 표현한다. (기하학적)

열벡터와 행벡터

벡터는 같은 모양을 가지면 element-wise 덧,뺏,곱셈이 가능하다.

- 덧셈을 기하학적으로 보면 다른 벡터로부터 상대적 위치 이동
- 뺄셈은 덧셈인데 이제 - 뒤에 나오는 벡터가 -된 즉, 방향이 반대로 된 벡터이고 그걸 더한 것과 같다. 즉, 뒤집은 덧셈

벡터의 노름
노름 = 원점으로부터의 거리
l1 노름: 각 성분의 변화량의 절대값 (택시 기하학)
l2 노름: 피타고라스 정리를 이용한 유클리드 거리 (유클리드 기하학)

언제 뭘 사용? 머신러닝의 각 피처들의 특징에 따라 적절한 것을 사용
l1: robust 학습, 라쏘 회귀
l2: laplace 근가, ridge 회귀

두 벡터 사이의 거리 l1 사용
||y-x|| = ||x-y||

두 벡터 사이의 각도 l2 사용
제 2 코사인 법칙 사용
자세한 건 수식으로 정리 (분자는 내적)

내적의 기하학적 의미
정사영된 벡터의 길이와 관련
내적은 정사영의 길이를 벡터 y의 길이||Y||만큼 조정한 값

## 행렬

행렬은 뭔가요?
벡터를 원소로 가지는 2차원 배열
행과 열이라는 인덱스를 가진다.
행렬의 특정 행을 고정하면 열벡터라 부른다. 반대로는 행벡터

표기법
벡터는 찐한 볼드체 X_1이면 1번째 행벡터
x_ij는 요소를 표현한다
X^T는 전치행렬을 뜻함

행렬의 기하학적 표현
벡터 = 공간에서 한 점
행렬 = 여러 점
그러므로 행렬의 한 행(=행벡터)은 i번째 데이터를 의미한다.
x_ij는 i번째 데이터의 j번쨰 변수의 값을 말한다.
선형대수로 접근하면 쉬움. 한 행이 방정식이고 요소 하나하나가 계수임.

행렬 덧셈, 뺄셈 - 같은 크기여야 한다.
성분곱 - 벡터와 동일(element-wise 곱) - 얘도 같은 크기
스칼라곱도 뭐 벡터랑 똑같음

행렬 곱셈
i번째 행벡터와 j번째 열벡터 사이의 내적을 성분으로 가지는 행렬을 계산
X의 열의 개수 == Y의 행의 개수
@ 연산자

내적
np.inner
i번쨰 행벡터와 j번쨰 행벡터 사이의 내적을 말한다.
X번째 행의 개수 == Y의 행의 개수

행렬을 기하학적으로 생각
행렬 = 벡터공간에서 사용되는 연산자
그림 가져오기
m차원의 벡터를 n차원의 벡터로 변환하는 것!
이로 패턴을 추출하거나 데이터를 압축할 수 있음

> 모든 선형변환은 행렬곱으로 계산할 수 있다!

역행렬
조건 1. 행과 열의 개수가 같고
조건 2. Determinant가 0이 아닌 경우
에만 역행렬이 존재한다.
np.linalg.inv

근데 역행렬이 없는 것들은? 슈도-역행렬
유사역행렬 / 무어-펜로즈 역행렬
n과 m의 크기에 따라 순서가 정말 중요함
이거는 더 공부해야함
np.linalg.pinv

역행렬 -> 방정식의 해를 구할 수 있음
슈도-역행렬 -> 선형회귀분석도 가능
이거는 다시 공부
