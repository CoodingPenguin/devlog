## Intro to NLP

nlp는 무엇인가?
자연어를 이해(nlu)와 자연어를 생성(nlg)를 목표로하는 분야
딥러닝이 진보하면서 떠오른 주요 분야 중 하나

예를 들면 language modeling, machine translation, question answering, document classification, diag system 등의 응용 분야가 있다.

nlp관련

유명 컨퍼런스 ACL, EMNLP, NAACL이 있고
최신 딥러닝 기술 기반 모델을 포함하고 있음

low-level parsing
토큰화: 문장을 의미가 있는 형태소 단위로 쪼개는 것 (형태소 = 토큰)
스테밍: 어근을 찾는 것 studied, studying -> study 이런식으로

word and phrase level

- ner: 단일 단어로 이루어진 고유명사 파악 ex. New York Times면 끊어서 하지 않고 한 번에 고유명사로 파악
- pos tagging: 단어 내의 품사나 성분이 무엇인지 알아내는 것 ex. 주어, 목적어, 동사 등
- noun-phrase chunking
- dependency parsing
- conference resolution

sentence level

- sentiment analysis: 감정 분석 (긍부정)
- machine learning: 기게번역 (어순을 잘 고려해야함)

multi-sentence and paragraph level (문단)

- entailment prediction: 두 문장 간의 논리적 모순단계 예측 (두 문장 사이의 모순이 있는지 파악)
- question answering: 질의 응답 (구글에 질문 검색하면 대답이 나오는 것)
- dialog systems: 챗봇 대용사항
- summarization: 문서를 한 줄로 요약

text mining관련

유명 컨퍼런스 KDD, The WebConf(WWW), WSDM, CIKM, ICWSM

빅데이터 분석 쪽
에를 들면, 특정 키워드 빈도수를 분석하는 것
회사 상품 출시해서 사람들의 후기를 수집해 반응을 분석

- 문서에서 유용한 정보나 인사트를 추출 ex. 트랜드 분석
- 문서 클러스터링 - 키워드 묶기 ex. 비슷한 무제로 데이터를 묶는 것
- 사회과학과 밀접한 관련 ex. 트위터 -> 신조어 추출

정보 검색 관련
네이버나 구글 검색 기술은 어느 정도의 수준에 왔음
사회과학과 관련
아직 활발히 연구되는 분야는 아님
추천시스템으로 발전 -> 개인화된 광고, 검색, 상품 추천 등

이게 왜 중요? 추천시스템이 없다고 생각해봐.
내가 검색하고 오 좋은데 더 찾으려고 할 때
내가 다시 검색하고 해야하잖아. 즉, 키워드를 내가 다 잡아야해.

nlp 동향

- 시퀀스 - 단어 - 벡터 즉, 단어를 벡터로 표현하는 기술 word2vwc glove 사용
- RNN 기반 모델사용 lstm, gru
- attention 모듈과 transfomer 모델로 성능이 많이 향상되었음 rnn이 대최가 된다.
- transformer로 기계학습 성능이 많이 좋아졌음.
- 원래는 nlp task별로 개별화된 모델이 필요해으나 transformer 이후로 스스로 학습하는 self-supervised training -> 일반화된 모델이 나옴! gpt, bert(근데 겁나 리소스가 크져)
- gpu 성능도 좋아야 하고 데이터셋도 커야 하고 이런 문제가 있긴함

## Bags of Words

딥러닝 적용 이전 단어를 표현한 방법

1. 유일 단어(중복 제거) -> vocabulary(사전)을 만듦
2. 각 유일 단어를 원-핫 벡터로 변환 (차원은 단어의 개수만큼)
   - 이 때, 모든 단어사이의 유클리다인 거리는 루트2
   - 코사인 유사도는 0임
   - 즉, 의미와 상관없이 동일한 연관성을 가지고 있음.

이렇게 만든 원-핫 벡터를 가지고 문장/문서를 표현할 수 있음
예시. 설명
이렇게 되면 각 단어는 인덱스로 인코딩되고 인덱스에 있는 숫자는 각 단어의 빈도수가 된다.

각 단어 별로 가방을 준비하고 특정 문장에서 나타난 단어들을 순차적으로 해당 가방에 넣어준 후 최종적으로 각 차원에 해당하는 가방의 단어 개수를 세서 벡터로 나타내는 방법이다! (요약)

예를 들면, cat과 kitty는 비슷한 벡터 표현을 가짐 (두 벡터간 거리가 짧음)
hanburger와 cat은 다른 벡터 표현을 가지고 두 벡터간 거리가 길다

워드 임베딩도 하나의 모델? 신경망이고 학습을 통해 단어를 최적화된 벡터로 표현하게 됨
