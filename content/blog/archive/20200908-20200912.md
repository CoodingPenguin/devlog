- **프로그래머스의 [비밀지도](https://programmers.co.kr/learn/courses/30/lessons/17681) 문제를 풀었다!**
  - `|` 연산자를 사용해서 간단히 풀 수 있는 문제이다. 이진수로 변환하는 `bin` 내장함수를 알게 되었다. 있을 것 같긴 했는데 `format` 함수의 형식지정자를 **b**로 해서 이진수로 변환하였다. 거기에 파이썬 문자열 함수가 얼마나 많은지를 봤는데 [W3School Python String Methods](https://www.w3schools.com/python/python_ref_string.asp)을 보면 알 수 있다. 내가 굳이 구현하지 않아도 메서드 하나면 충분히 할 수 있는 그런 메서드들이 많았다. 한 번 싹 정리해야겠다. (정리해야할 게 너무 많다ㅠㅠ)
- **[Fruits-360 - Transfer Learning using Keras](https://www.kaggle.com/amadeus1996/fruits-360-transfer-learning-using-keras) 1차 필사를 마쳤다.**
  - 이번에는 이미지 다중 분류를 하는 것인데 이 데이터가 너무 커서 설계한 모델을 돌리지 못했다. 그래서 이 노트북은 자율로 하기로 했다. 그리고 지금 모듈 간 Dependency도 다 꼬여있어서 *Keras*가 잘 실행이 안 된다ㅠㅠ 전이 학습(Transfer Learning)에 대해 배웠는데 **다른 데이터셋으로 학습한 모델을 가져와(=가중치를 가져와) 학습시키는 것**이다. 근데 우리가 사용하는 데이터셋은 모양이나 크기도 다를테니 그거에 맞추어 마지막의 완전연결계층을 데이터셋에 맞게 변형해준다! 거기 댓글을 보면 정확도가 98%가 나오는 걸 보니 효과는 좋은 것 같다.
  - 원래 가상 환경을 만들지 않고 하는 경우가 많았는데.. 써보니 가상환경의 필요성을 뼈져리게 느끼고 있다. `requirements.txt` 모음집을 만들어놓거나 아에 Docker를 배워서 `도커 이미지`를 사용해야겠다. 불편함이 생기니 배워야할 게 생긴다.
