## 문제

BOJ 1904번 : [01타일](https://www.acmicpc.net/problem/1904)

## 접근 방법

`00` 또는 `1`인 타일로 만들 수 있는 타일 조합을 구하는 문제이다. 만들고 싶은 타일의 개수가 $N$일 때의 조합의 개수는 **이전 타일의 조합에 `00` 혹은 `1`을 붙여서 만들었을 때의 조합의 개수**이다.

### 설명

예를 들어, **$N=3$일 때의 조합**을 만들어보자. $N=1$일 때 가능한 조합은 $1$이고, $N=2$일 때 가능한 조합은 `00`, `11`이다. $N=3$일 때의 조합은 $N=1$일 때의 조합에 타일 `00`을 붙이거나 $N=2$일 때의 조합에 타일 `1`을 붙이는 것이다.

- $N=1$일 때 : `100` `001`
- $N=2$일 때 : `100` `001` `111`

하지만 중복된 타일이 있으므로 $N=3$일 때 가능한 조합의 개수는 **3개**이다.

**$N=4$일 때**도 동일하다. $N=2$일 때의 조합에 `00`을 붙이거나 $N=3$일 때의 조합에 `1`을 붙이면 된다.

- $N=2$ 일 때 : `0000` `1100` `0011`
- $N=3$ 일 때 : `1001` `1100` `0011` `1111`

이 때도 중복이 있으므로 중복을 제외하면 가능한 조합의 개수는 **5개**이다.

### 결론

계속 타일의 개수를 늘리다보면 현재 만드려는 조합과 그보다 타일의 개수가 적은 조합의 관계를 **점화식**으로 나타낼 수 있다. 타일의 개수가 $i$개일 때의 만들 수 있는 조합의 개수를 $comb[i]$라 할 때 점화식은 다음과 같다.

$$
comb[i] = comb[i-1] + comb[i-2]
$$

이렇게 도출한 점화식을 그대로 코드로 옮겨주면 된다!

## 교훈

지금까지의 동적계획법 문제풀이로 봤을 때 핵심은 **점화식 도출**이다!

## 소스 코드

```python
n = int(input())
result = [1]*(n+1)
for i in range(2, n+1):
  result[i] = (result[i-1] + result[i-2])%15746

print(result[n])
```
