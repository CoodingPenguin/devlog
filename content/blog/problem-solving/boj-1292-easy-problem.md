## 문제

BOJ 1292번 : [쉽게 푸는 문제](https://www.acmicpc.net/problem/1292)

## 접근 방법

### 방법 1.

가장 처음 생각했던 방법인데.. 여기서 가장 중요한 것은 바로 **시작과 끝에 해당하는 숫자**와 **해당 숫자가 연속된 숫자(ex. 55555)의 수열에서 몇 번째인지**였다. 예를 들어, 시작이 3이면 해당되는 숫자는 2이고 연속된 2의 수열에서 두 번째이다. 이를 `my_number` 함수로 구현하였다! 3을 입력으로 넣으면, (2, 2)인 튜플이 반환된다.

반환된 튜플을 가지고 두 가지 경우로 나눌 수 있다.

**첫 번째는 시작과 끝의 튜플이 같은 경우이다. 예를 들어, 시작과 끝이 반환된 튜플이 (5, 2), (5, 2)로 동일한 경우다.**

- `my_number`라는 함수를 통해 시작(or 끝)의 해당 숫자를 구해서 출력한다.

**두 번째는 시작과 끝의 튜플이 다른 경우이다. 예를 들어, 시작이 (5, 1)이고 끝이 (6, 4) 처럼 반환된 튜플이 전혀 다른 경우이다.**

- 시작과 끝을 `my_number` 함수에 넣어 시작과 끝의 **(해당 숫자, 해당 숫자에서 몇 번째인지)**를 각각 구한다.
- 시작과 끝의 해당 숫자가 같은 경우. 예를 들어, 시작이 (5, 1) 끝이 (5, 4)인 경우 5를 1, 2, 3, 4 즉, 4번 거쳐가니 **5\*4**를 출력된다. 이를 수식으로 나타내면 **(해당 숫자) \* (끝의 해당 숫자의 순서 - 시작의 해당 숫자의 순서 + 1)**이다.
- 시작과 끝의 해당 숫자가 다른 경우. 예를 들어, 시작이 (3, 3) 끝이 (6, 2)인 경우 해당 숫자인 3과 6을 제외하고 그 사이 값들의 수열 합인 **4\*4+5\*5**를 구한다. 그리고 시작과 끝에 해당하는 숫자에서 몇 번 거쳐가는지를 구한다. 즉, **3\*1+6\*2**이다. (수식은 코드를 참고하세요!)

<br>

`my_number` 함수를 설명하자면 1부터 x까지의 합인 $\frac{x(x+1)}{2}$를 이용해 구현했다. 만약 합이 $3$이면, $\frac{2(2+1)}{2} = 3$이 되어 2까지 더한 수라는 걸 알 수 있다. 즉, 저 $x$를 구하면 **해당하는 숫자**를 구할 수 있다. 입력이 n일 때 $\sqrt{2*n}$을 이용해 근사적으로 저 $x$을 추측했다. 하지만 우리가 추측하는 값보다 적게 나오기 때문에 이를 보완해야 한다.

그래서 근사적으로 구한 값까지의 합. 예를 들어, 결과가 5가 나왔다면 **1+2+3+4+5**을 일단 입력값에서 뺀다. 그리고 반복문을 돌리면서 입력이 0보다 크거나 같으면 6을 빼보고 그래도 괜찮으면 7을 빼본다. 그러다 0보다 작아지면 해당 숫자를 이미 지났다는 것이므로 반복문을 빠져나오게 된다.

### 방법 2.

위의 방법은 좀 많이 복잡하고 메모리를 조금 잡아먹어서 다른 분들의 풀이를 찾아봤는데 [이 분의 풀이](https://www.acmicpc.net/source/17786361)가 매우 간단해 이를 소개해보려 한다. 일단 여기서 가장 중요한 것은 **입력의 범위**이다! `입력의 범위`는 **1부터 1000**까지이다. 이를 통해 추론할 수 있는 것은 `해당되는 숫자`가 **1부터 45(1+...+45=1035)**이다.

이를 이용하여 리스트를 생성하여 **[1, 2, 2, 3, 3, 3, .... , 45, 45, 45]**로 만든다. 그러고 시작과 끝 값을 입력받아서 **슬라이싱**을 해준 뒤 **합**을 구하면 금방 구할 수 있다!

## 교훈

이 문제에서 얻을 수 있는 교훈은 **"입력의 범위를 무조건 확인해야한다"**이다! 이것으로 코드 몇 줄은 줄일 수 있다. 그리고 두 번째 방법을 보면 **리스트를 생성**한 뒤 추가 리스트를 생성하지 않아서 오히려 메모리 절약이 되는 걸 볼 수 있다. 전에 풀었던 문제에서도 비슷한 방법이 쓰인 적이 있는데 **입력 범위가 적고 알고리즘이 복잡**해지면 한 번 시도해봐야 겠다.

## 소스 코드

### 방법 1.

```python
# 풀이 1
import math


def add_all(limit):
    return sum(list(range(1, limit+1)))


def my_number(n):
    m = int(math.sqrt(2*n))
    n -= add_all(m)
    while True:
        if n <= 0:
            break
        m += 1
        n -= m
    return m, (n+m)


def result(s, e):
    ss, ee = my_number(s), my_number(e)
    rt = 0
    if ss[0] == ee[0]:
        rt += ss[0] * (ee[1] - ss[1]+1)
    else:
        for i in range(ss[0]+1, ee[0]):
            rt += i*i
        rt += (ss[0] - ss[1] + 1) * ss[0] + ee[1] * ee[0]
    return rt


_ = list(map(int, input().split()))
s, e = _[0], _[1]

if s == e:
    print(my_number(s)[0])
else:
    print(result(s, e))
```
